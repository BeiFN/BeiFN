```javascript
    function foo(o){
        // 1. 先声明提升  var o;
        // 2. o 形参被 实参赋值 ;
        // o ==> 实参;
        console.log(o)
        o.a++;
        var o = {a : 20} // 对象②
        o.a ++;
    }
    var obj = {a : 10}; // obj 存的是地址; 对象①
    foo(obj);
    console.log(obj);            这个不太懂
```

>  声明提升优先于实参给形参赋值 ,如果同时存在声明提升和实参赋值的话,那么实参赋值有效。
