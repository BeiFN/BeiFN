
## 接口文档

- 路径      : http://localhost/gp12coding/0712/login.php

- 请求方式   : POST 

- 参数(字段) : 

      username:"username",
      password:"password" 

- 返回值     : 
      ```
            {
                  state : "success|error",
                  stateCode : 0 , 
                  [,username : 用户名  [,password : md5加密后的密码]]
            }
      ```
      stateCode : 0 缺少参数;
                  1 表示登陆成功;
                  2 数据库连接失败;
                  3 用户名不存在;
                  4 密码错误;
