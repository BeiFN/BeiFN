10. ```javascript
    function foo(o){
        // 1. 先声明提升  var o;
        // 2. o 形参被 实参赋值 ;
        // o ==> 实参;
        console.log(o); // {a:10}
        o.a++;
        var o = {a : 20} // 对象2
        o.a ++;
    }
    var obj = {a : 10}; // obj 存的是地址; 对象1
    foo(obj);
    console.log(obj); // {a:11}
    ```

    > 声明提升优先于实参给形参赋值 ,如果同时存在声明提升和实参赋值的话,那么实参赋值有效。

    > "a"  in  window;  window之中是否存在a属性;

    > **属性** 对象之中的key;
    >
    > **变量** 使用var声明的;

    > 声明全局变量时, 就想相当于 给 window 添加一个变量名的属性;
    >
    > var b = 10;     window.b = 10;
    >
    > "b" in window // true;
