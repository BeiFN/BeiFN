# 事件
1.事件流 ，事件执行顺序
    *事件流：事件传播的过程
    *事件触发阶段
        1.捕获
        2.目标
        3.冒泡
    *三个阶段分成两种情况
        1.IE：先目标再冒泡
        2.netspace 先捕获再目标

2. **事件监听**
    *什么是监听？和onclick有啥区别
    *是在捕获阶段或者冒泡阶段添加监听方法的一个工具
    *addEventListener(事件名，事件处理函数， boolean[捕获|冒泡false默认])
    *removeEventListener
    *封装 on off(事件名，事件处理函数)

3.事件流，事件监听能力
    **阻止事件冒泡**
    *stopPropagation
    ```javascript
        typeof e.stopPropagation === "function" ? e.stopPropagation() : e.cancelBubble = true;
    ```


4. **拖拽**
    * 鼠标按下  拖拽开始
    ```javascript
        // 获取offset值
        e.offsetX
        e.offsetY
    ```
    * 鼠标移动  拖拽进行
    * 鼠标抬起  拖拽结束
