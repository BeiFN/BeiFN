##问题4  能不能给咱整整这句话的意思
```javascript
      jQuery.extend = jQuery.fn.extend = function(){
      //.....
      }
      // 
```

* 给两个属性进行赋值赋值为同一个函数。
* extend 方法 期中有一个功能 直接向构造函数或原型合并对象。
* 就是在这个函数调用的时候想要有两种this;
```
      $.xxx    this => 构造函数;
      $().xxx; this => 实例对象;
```
