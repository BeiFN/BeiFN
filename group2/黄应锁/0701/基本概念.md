    构造函数和普通的函数有啥区别?
    调用方式不同： 普通函数：函数名（）；构造函数： new 函数名（）；这样的构造方式去创建对象；

构造函数   原型     实例    的关系：
构造函数和原型的关系：伴生 ：每一个构造函数都有一个与之对应的原型。
构造函数中用来存放属性；原型中用来存放函数方法；
构造函数： function Foo（）{   }
原型     ：   Foo.prototype.函数名 = function （）{ }

构造函数在调用的时候都会创建一个新的实例对象。
