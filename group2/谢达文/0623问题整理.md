问题3：var a之后，为什么a in window 和 "a" in window 得到的结果都是true，二者有什么区别吗？
解答： in 关键字: 判定某个属性是否存在于对象之中。
       "a"  in  window;  window之中是否存在a属性;

        **属性** 对象之中的key;
        **变量** 使用var声明的;

        声明全局变量时, 就想相当于 给 window 添加一个变量名的属性;
        var b = 10;     window.b = 10;
        "b" in window // true;
